我们收集的信息

我们以多种方式收集数据以改善我们的产品和服务：

  • 您提供给我们的信息
    这包括您在注册和设置新产品、下载软件、在论坛或社区等在线平台上发布、回复调查或参加比赛时提供的信息。
  • 使用产品和服务的信息
    当您使用产品和服务时，可能会向我们发送数据，如您的硬件型号、固件和操作系统版本、唯一设备标识以及 IP 地址/位置信息。有时，在遵守适用法律并需要您同意的情况下，我们可能会根据您的活动收集数据，包括您的设置和配置、使用情况数据以及崩溃或错误数据。

我们还可能会在开展业务和履行协议时收集和以其他方式处理您提供给我们的或我们从贵组织（如我们的业务合作伙伴和供应商）处收集的业务联系信息。如果此类处理需要通知或同意，则组织将通知并获得其各自人员和授权用户的此类同意。

我们收集的数据内容取决于您使用的产品、功能和服务，以及您可能参与的活动、业务往来和其他您可能与我们进行的互动。此类数据可能包含以下内容：

  • 姓名和联系数据 – 您的姓名、个人或公司电子邮件地址、个人或公司邮寄地址、个人或公司电话号码、用户 ID 和其他类似联系数据等数据。
  • 证书 – 用于身份验证和帐户访问的密码、提示和信息
  • 人口统计数据 – 诸如您的年龄、性别、国家和首选语言之类的数据
  • 付款数据 – 您通过罗技网站进行消费所需的处理您订单的所需信息，如配送地址、所订购产品以及联系信息。当您购买数字产品时，我们会处理支付数据，例如交易价值、您购买的订阅内容，以及您的 Paypal、WeChat、AliPay 或 Stripe ID（如适用）。我们不会收集支付信息或信用卡号码。
  • 使用数据 – 您和您的设备与我们以及我们的产品进行互动的相关数据，例如：
    • 产品使用收据 – 您使用的功能、采取的操作以及执行的活动的相关数据，例如 按键操作、与应用程序的交互或者使用扬声器的时长
    • 设备数据 – 您的设备和用于连接到我们产品的网络相关数据，包括设备的操作系统和其他软件、设备标识符以及区域和语言设置；
    • 其他产品数据 – 来自其他设备和软件的信息，例如立体声、家庭自动化和电视产品信息、电缆提供商信息
    • 错误报告和性能数据 – 问题的类型和严重程度、错误相关软件或硬件的细节、错误发生时您正使用的文件内容等数据，以及您设备上其他软件的相关数据。
    • 支持数据 – 有关您和您的硬件、软件的信息，以及有关事件的详细信息、身份验证数据、您与我们的通信、有关您所使用的计算机和应用程序的数据以及系统、注册表和配置数据。
  • 兴趣和偏好 – 您的音乐喜好、计划活动或例程、频道和广播电台以及您使用的服务
  • 联系人和人际关系 – 如果您使用支持多用户、社区和共享功能的产品，我们可能会收集有关您的联系人和人际关系的数据
  • 罗技帐户（访客和 Logi ID） – 有不同的选项可用于创建帐户，具体取决于产品或服务以及您的选择。您可以决定不创建帐户，但以访客身份提供详细信息。在此类情况下，我们可能只收集有限的信息，例如您的姓名、电子邮件地址和送货地址。您可以决定在我们的网站上创建一个名为 Logi ID 的个人罗技帐户。在此情况下，我们还可能收集您的姓名、电子邮件地址、送货地址等信息，这将优化您的访问体验。例如，可以启用更个人化的个性化设置，并允许您保存和调用设置、配置和历史记录。我们将为您的帐户和信息分配一个唯一的 ID 号。当您登录帐户时，我们可能会收集与您会话相关的信息，包括登录记录、日期和时间、您登录的产品、您帐户的编号以及您设备的唯一标识、IP 地址和您的操作系统和浏览器版本。通常状态下，登录帐户后您将保持登录状态，直至您登出账户。
  • 社交登录 – 当您使用第三方服务（如 Apple、Facebook、Google 或 Twitch）登录时，我们会处理您已同意通过相关第三方平台共享的个人数据。请在关联您的帐户时查看这些第三方服务向您提供的隐私声明，了解有关罗技收到的信息的详情。
  • 位置数据 – 一些产品会出于时间/日期和语言偏好等目的或者法律合规目的而使用位置数据。我们可能需要收集的位置数据精度不一。精确位置数据可能是您的家庭或工作地址信息或全球定位系统数据，我们可能在您启用基于位置的产品或功能时收集此类数据。非精确位置数据可能包括您的 Wi-Fi 或 IP 地址代表的位置信息，或者指出您所处位置大致区域的低精度数据（如所处城市或地区）。
  • 内容 – 我们可能会在必要时收集您的文件和通信内容相关信息，以向您提供您所使用的产品。我们同样可能会收集您发送给我们的信息的内容，例如您撰写的反馈和产品评论，或者您向客户支持提交的问题和信息。您与我们联系时，与代表的电话交谈或聊天会话可能会在遵守适用法律的情况下，经您同意后进行监控和录音。

我们可能会在您同意的情况下共享您的个人数据，或为了完成交易，或者提供您申请或授权的产品或服务。出于本政策中所述的目的以及法律要求，我们还将在我们的受控分支机构和子公司之间共享个人数据。

若您在我们产品或服务的公共区域（如社区、论坛或博客）发布信息，这些信息将面向公众开放，并且我们可自行决定并按“使用条款”对此类信息进行再利用或再发布。

最后，在我们为实现下述目的而确信有必要的情况下，我们将访问、转让、披露和保存包括您的内容在内的个人数据：

  • 遵守法律或回应法律程序，例如来自执法机构或其他政府机构的法律程序；
  • 保护客户，例如防止垃圾邮件或企图欺诈我们用户的欺诈邮件，或协助防止人身伤害；
  • 维护我们产品的安全性，包括防止或阻止对我们系统或网络的攻击；
  • 保护您或我们的法律权利，或根据法律或法院或政府机构命令的要求。

备份和传输设备设置

Logitech Options 允许您自定义计算机的鼠标、键盘和触控板。Logitech Capture 软件使您可以在计算机上录制视频和音频并更改摄像头的设置。该软件还让您可以将设备设置备份至云中，从而将这些设置转移到其他计算机并轻松设置设备。

可能收集的信息包括：

  • 创建账户或者使用第三方开放式授权 (OAuth2) 登录（如 Facebook 或 Google）所需的帐户信息（例如名称、电子邮件地址和密码）。
  • 计算机信息，例如计算机的名称、制造商和型号，以确保该设备可以正常工作。
  • Logitech Options 的特定信息 -
    • 按钮和按键自定义，包括特定于应用程序的自定义
    • 其他设备设置，例如鼠标的指针和滚动设置
    • 分析数据是在您同意的情况下收集的。
  • Logitech Capture 的特定信息 -
    • 场景组件和摄像头设置

使用 Logitech Options 时，设备设置将自动备份。您可以选择不自动备份或注销设备设置，从而有效地关闭备份功能。

软件更新

我们的服务器会在安装软件期间和之后自动记录某些信息，包括 IP 地址、软件版本和连接的设备，以检查是否需要进行必要的软件更新并下载所需的设备影像或资产。

性能、诊断和分析

在您同意的情况下，我们的某些软件（如 Logitech Options、Capture、Presentation 应用程序、Setpoint 和 Logitech Control Center）会收集并记录有关该软件、设备的信息以及其他信息，例如

  • 软件功能的使用
  • 设备的自定义设置
  • 系统硬件和软件，如操作系统版本
  • 使用 IP 地址的城市级位置信息
  • 设备使用频率，包括点击次数和按钮计数。例如，为分析产品耐用性和可靠性，我们可能捕获每个按钮在 24 小时内的使用次数。
  • Logitech Capture 的特定信息 – 我们捕获场景使用情况和摄像头型号信息。我们不会分析或传输用户的音频或视频。
  • Logitech Presenters 会在您开始和结束演示时捕获您所使用的应用程序名称和版本。我们收集此信息，以便帮助改进功能。

我们不会收集您在使用鼠标、键盘或触摸板时输入的数据。按键分配是指当您自定义设备并分配快捷键时，我们的应用程序需要您的许可才能执行分配的快捷键。

产品功能

我们有时会收集设备和应用程序信息等数据，以使产品功能正常运作。例如，Logitech Flow 是某些设备中的功能，让您可以在两台或三台计算机之间顺畅切换，并将文本、图像和文件从一台计算机传输到另一台计算机。为在 Flow 的设置过程中自动发现计算机，我们可能会收集您的设备标识符和 IP 地址。
